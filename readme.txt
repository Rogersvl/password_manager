# 🔐 Password Manager

Um gerenciador de senhas simples com interface gráfica feita em **Tkinter** e persistência de dados usando **SQLAlchemy** com SQLite.

---

## 📸 Preview

![screenshot](screenshot.png) <!-- você pode adicionar um print da aplicação aqui -->

---

## ✨ Funcionalidades

- Adicionar senhas com nome do site, e-mail e senha
- Buscar senhas salvas por nome do site
- Geração automática de senhas seguras
- Dados salvos localmente em banco SQLite (`passwords.db`)

---

## 🧰 Tecnologias usadas

- Python 3
- Tkinter
- SQLAlchemy 2.0
- SQLite
- JSON (para geração de senhas)
- Boa prática com tipagem `Mapped[]` e `DeclarativeBase`
